# next-nice-comments
> Build nice comments.

## install:
```bash
npm install -S afeiship/next-nice-comments --registry=https://registry.npm.taobao.org
```

## usage:
```js
var comments = [
  'name: next-nice-comments',
  'description: Just a lite',
  'licence: MIT'
];
nx.niceComments(comments,'js');
nx.niceComments(comments,'html');
```
